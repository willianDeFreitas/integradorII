Lista 1 - Fundamentos de Web Design 1
1- Qual a forma correta de importarmos um arquivo CSS para nossa página HTML?

a)link rel="stylesheet" href="css/arquivo.css"
b)link rel="stylesheet" css
c)stylesheet="css" href="css/arquivo.css"
d)Stylesheet="css" inport="css"

2- Qual tag devemos aplicar dentro da tag head para podermos fazer uma alteração de CSS?

a) mudar
b) style
c) css
d) alterar/css

3- Qual a sintaxe para aplicação do CSS in-line dentro de um parágrafo?

a) p style "frase aqui"
b) p style="css" mudar: "red" "frase aqui" p
c) p style="color:red" "frase aqui" p
d) nenhuma das alternativas acima

4- Qual a sintaxe de uma tag style?

a) seletor{propriedade:valor;}
b) class{valor>propriedade}
c) seletor:valor(propriedade)
d) propriedade:valor = css

5- Qual formatação terá prioridade sobre todas as outras, caso haja importação, in-line e na tag head?

a) in-line
b) @inport arquivo css
c) inportação de arquivo css
d) tag style
